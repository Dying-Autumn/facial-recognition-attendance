# 项目启动说明

## 启动项目

在当前文件夹下打开终端，输入以下命令启动项目：

```bash
mvn spring-boot:run
```

启动后，在浏览器访问 `http://localhost:8080`

**注意：需要电脑已安装 Maven**

## 停止项目

如果项目正在运行，需要先停止才能重新启动。有两种方式：

### 方式1：使用停止脚本（推荐）

```bash
./stop.sh
```

### 方式2：手动停止

1. 查找占用8080端口的进程：
```bash
lsof -ti:8080
```

2. 停止进程（将PID替换为实际的进程ID）：
```bash
kill -9 <PID>
```

## 重启项目

使用重启脚本（会自动停止旧进程并启动新进程）：

```bash
./restart.sh
```

或者手动操作：
1. 先执行 `./stop.sh` 停止项目
2. 再执行 `mvn spring-boot:run` 启动项目

## 常见问题

**问题：端口8080已被占用，无法启动**

**解决方案：**
- 使用 `./stop.sh` 脚本停止占用端口的进程
- 或者手动查找并停止进程：
  ```bash
  # 查找进程
  lsof -ti:8080
  
  # 停止进程（替换PID为实际进程ID）
  kill -9 <PID>
  ```
